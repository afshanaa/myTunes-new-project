The MyTunes music player is a straightforwardz,
command-line-based application designed to provide basic music playback functionalities using Python
and the pygame library. The player initializes the pygame.mixer module to handle audio playbackensuring
that users can listen to their favorite .mp3 files fro a specified directory Upon starting the application,
users are presented with a menu offering four primary options:listing available songs, playing a selected song,
stopping the current playback, and exiting the program. The MyTunes class encapsulates the core functionality,
including methods to list all songs in the designated music folder, play a song by loading and playing the selected file via pygame
mixer.music.load and pygame.mixer.music
